Overview

Designed and implemented the network and communication infrastructure for a restaurant, including a Cisco router, 4 internet routers, switches, and a server. The project covered structured cabling, network connectivity, Wi-Fi coverage, IP telephony, and printer distribution across all departments, and integrated secure branch connectivity using SoftEther VPN.

My Role

  • Collaborated with my team to configure Cisco switches and router.
  • Independently configured and installed all Cisco CP-7942 IP Phones across restaurant sections, enabling seamless internal communication.
  • Installed a PVDM module in the Cisco router to support voice processing and IP telephony services.
  • Assisted in configuring and deploying 5 Cisco Access Points to ensure full Wi-Fi coverage across the restaurant.
  • Independently configured and distributed all HPRT network printers, including driver installation and integration with cashier PCs.
  • Contributed to the configuration of the main internet router and other internet routers to match our requirements.
