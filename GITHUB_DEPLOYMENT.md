# 🚀 GitHub Pages Deployment

## Quick Setup

1. **Repository:** [https://github.com/yossef-salem/devjoex](https://github.com/yossef-salem/devjoex)
2. **Enable Pages:** Settings → Pages → Source: "Deploy from a branch" → Branch: "main"
3. **Wait:** 5-10 minutes for deployment
4. **Visit:** [https://yossef-salem.github.io/devjoex](https://yossef-salem.github.io/devjoex)

## Deploy Updates

```bash
git add .
git commit -m "Update portfolio"
git push
```
