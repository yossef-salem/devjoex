# 🚀 DevJoex Portfolio

A modern, responsive personal portfolio showcasing DevOps and IT engineering skills.

## ✨ Features

- **Responsive Design** - Works perfectly on all devices
- **Modern Animations** - Smooth, professional animations
- **Contact Form** - Integrated with Formspree for email notifications
- **Auto-resizable Textarea** - Dynamic contact form experience
- **Mobile Optimized** - One-row tabs on mobile devices
- **SEO Friendly** - Optimized for search engines

## 🛠️ Technologies

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with animations
- **Vanilla JavaScript** - Lightweight and fast
- **Formspree** - Contact form backend
- **GitHub Pages** - Free hosting
